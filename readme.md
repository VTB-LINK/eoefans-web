# eoefans-web 端

## 目录结构（暂定）

1. component：存放组件
2. utils：工具库
3. store：数据源
4. routers：路由

## Build Setup

```bash
# install dependencies
$ npm install

# serve with hot reload
$ npm run dev

# build for production
$ npm run build

# local preview
$ npm run preview
```
